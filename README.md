# Co-Deep Project Frontend

`npm start`로 실행됨.

실행 경로 : `cd frontend`

## Co-Deep Project backend

- python 가상환경 실행 `source myenv/bin/activate`
- FastAPI 서버 실행 명령: `uvicorn backend.server:app --reload`
